# akita-dot 🐕
My own custom rice of Hyprland based on the wallpapers colors!

# Installation
1. Apply **chmod +x** to `install_dependencies.sh* and **install_files.sh**.
2. Run **install_dependencies** to install required packages.
3. Finally, run **install_files.sh** to install the config. 

## Important ⚠️
- **Installing the config will delete your previus .config folder. Please consider doing a backup first!**
- **Don't forget to edit the env variables (.config/uwsm/env) and monitor config (.config/hypr/monitors.conf) to adjust your experience based in the hardware you have**
